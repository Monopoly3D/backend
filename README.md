# Packet Classification

### BasePacket

```json
{
  "data": {},
  "meta": {
    "id": "packet_id",
    "tag": "packet_tag",
    "class": "packet_class"
  }
}
```

Packet meta is a list of values which both client and server use to authorize and verify incoming packets, and to provide different implementations to process them
